# Изменения

## 2026-07-20

- Удалена автоматическая замена выбранной Qwen-модели на `qwen3-vl-plus` при отправке изображений.
- `OpenAiCompatibleProvider` теперь передаёт выбранный идентификатор модели без изменений и сохраняет Qwen upload в `files` последнего пользовательского сообщения.
- Добавлены контрактные тесты для `qwen3.8-max-preview` с изображением и без него.
- Добавлен диагностический лог выбранной и фактической модели перед отправкой запроса.
- FreeQwenApi: `qwen3.8-max-preview` больше не алиас `qwen3.7-max` — отдельная каноническая модель в `modelMapping.js`.
- `FixQwenProxyModelsAsync` теперь снимает 3.8 с чужих alias-групп и добавляет собственную; фикс применяется автоматически при старте прокси.
